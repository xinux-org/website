### Oyna Server

Biz o'z paketlarimizni yaratdik va uni o'z oyna serverimizda saqladik. Bizning oynamizdan foydalanish va paketlarimizni o'rnatish uchun siz GPG ochiq kalitini yuklab olishingiz va o'rnatishingiz kerak.

```shell
curl -Lo - "https://raw.githubusercontent.com/uzinfocom-org/keyring/main/xinux.gpg" \
    | pacman-key -a -

pacman-key --lsign-key info@uzinfocom.uz

pacman-key --init
pacman-key --populate xinux
```

keyin bizni mirrorlistimizni `/etc/pacman.conf` ga quyidagicha qo'shib qo'ysangiz bo'ladi:

```yaml
[xinux]
SigLevel = Required DatabaseOptional
Server = https://mirror.xinux.uz/$repo/$arch/
Server = http://mirror.dc.uz/$repo/$repo/$arch/
Server = https://mirror.dc.uz/$repo/$repo/$arch/
```
